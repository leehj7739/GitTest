카카오 클라우드를 활용하여 2-tier 구성
- 클라이언트/서버
- db 서버

1. 로컬 환경에서 테스트 및 도커 이미지를 도커 허브에 push


2. 카카오 클라우드 인스턴스 생성 및 기본 세팅
- app	> 포트:80
- db	> 포트:3306

3. 카카오 클라우드 환경에서 도커 이미지 pull & run
- app  > app 이미지 pull & run
- db   > db 이미지 pull & run

4. 카카오 클라우드 보안그룹 인바운드 규칙 확인
- 이미 80, 3306 포트 열린것을 확인

5. http://ip:80 접속
- 데이터베이스 연동 확인
- 조회, 추가, 삭제 테스트 
> 삭제 코드 작성 (chat gpt, 코드 내용 검수)

6. 그라파나 설정
- mysql <-> 그라파나 데이터 연결
- 대시보드 작성
1. 추가된 사용자 (누적)
2. 현재 사용자 숫자
