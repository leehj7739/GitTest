네이버 블로그 검색 api 연동하여 카카오 클라우드에 서비스 올리기
1. 네이버 디벨로퍼스 
- 로그인
- 어플리케이션 등록 > 검색
- 검색용 id, secret 획득 ( 보안이슈가 있으나, 검색량 제한이 있어 그냥 업로드)
           'X-Naver-Client-Id': 'Wr40DDBa3DiBuxos2tsN',
           'X-Naver-Client-Secret': '9s9klEIvxH'

2. docker 기반 컨테이너화를 위해 프로젝트 구성

- 네이버 디벨로퍼스 예제 + chat gpt (html, css 등) 이용하여 프로젝트 구성 및 코드작성

MYTEST
|   app.js // Node.js 기반 앱의 시작점, 위의 id, secret 입력, 네이버 디벨로퍼스 예제 사용
|   Dockerfile  //도커 
|   package.json // Node.js 관리
+---public
|       style.css  //유저 인터페이스
+---routes
|       routes.js //서비스 경로 설정을 편하게 하기위해 ip:port/ 루트 위치에서 검색하도록 경로 추가
\---views
        index.html  //유저 인터페이스

3. 도커 이미지 빌드 및 도커허브 push
- 별 이슈 없음

4. 카카오 클라우드 환경에서 pull & run'

- 이슈 사항 1. 5000:3000 번으로 포트를 맵핑하였는데,  인바운드 규칙에 5000번이 없어 접근이 불가
> 인바운드 규칙에 5000번 추가 
> TCP	0.0.0.0/0	5000 		myapp-test  	2025.04.04 (금) 16:46
> http://myip:5000/  접속
> 이후 클라우드 환경 테스트시  접속 및 검색이 잘 작동함을 확인
